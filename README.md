# skinnyjeans
## A hipster implementation of WDM. 


This work in progress will leverage all of the latest and probably not the greatest technologies that everyone seems to lust after.

These include:
* React
  * Redux
  * Flow
  * API Sauce
  * Redux Sauce
  * GraphQL
  
* Node middleware

* Java 8
  * Spring 4
  * SpringRest
  * Docker
  
* MongoDB
